Kanban Adaptado

React
TanStack table
drag-and-drop
React-CSV
NodeJs
Express
PostgreSQL
Este projeto é um sistema de gerenciamento de fluxo de trabalho, no estilo Kanban, projetado especificamente para uma oficina automotiva. A plataforma permite visualizar o andamento dos reparos dos veículos em colunas dinâmicas, que podem ser criadas, editadas e reordenadas diretamente pela interface. Cada veículo é representado por um card que pode ser movido entre as etapas do processo. O sistema inclui uma tela separada para veículos concluídos, com filtros avançados e a funcionalidade de exportar os dados para CSV.

Bot Orçamentos

Python
Selenium WebDriver
Pandas
Este projeto automatiza o download de orçamentos de veículos em formato XML, otimizando um fluxo de trabalho que envolve múltiplos sistemas. O robô inicia lendo uma lista de placas de um arquivo CSV, processando-as primeiro no sistema Cilia. As placas não encontradas são subsequentemente pesquisadas no sistema Soma. Ao final do processo, o robô organiza os arquivos baixados e gera um relatório com as placas que não foram localizadas em nenhuma das plataformas, garantindo um controle completo e eficiente.
